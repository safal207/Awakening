# ТЕХНИЧЕСКОЕ ЗАДАНИЕ: Voxel PixelArt Engine

## 1. Концепция
Воксельный мир из кубов, где каждый блок — куб, 
а каждая его грань — это сетка 8×8 цветных пикселей (64 пикселя на грань).

## 2. Архитектура

### 2.1 Блок (Block)
- Форма: куб 1×1×1
- 6 граней: Up, Down, North, South, West, East
- Каждая грань: 8×8 пикселей (RGB)
- Прозрачные блоки (вода, листва) — альфа-тест
- Типы блоков: Air, Grass, Dirt, Stone, Wood, Leaves, Sand, Planks, Bricks, Water, Glass

### 2.2 Чанк (Chunk)
- Размер: 16×64×16 блоков
- Меш: для каждого видимого блока генерируются только открытые грани
- Каждая грань = 64 пикселя = 64 маленьких квадрата
- 1 квадрат = 2 треугольника = 6 вершин
- Всего на блок: до 6×64×6 = 2304 вершины (с оптимизацией — меньше)
- Dirty flag: меш перестраивается при изменении блока

### 2.3 Мир (World)
- Процедурная генерация через Perlin noise
- Чанки подгружаются вокруг камеры
- Разрушение (ЛКМ) и установка (ПКМ) блоков
- Рейкаст на 8 блоков

### 2.4 Рендер
- OpenGL 3.3 Core
- Цвета на геометрии, без текстур
- Направленное освещение + ambient
- Цикл дня/ночи

### 2.5 Управление
- WASD + Space/Ctrl — перемещение
- Мышь — осмотр
- Shift — ускорение
- ЛКМ — разрушить блок
- ПКМ — поставить блок (тип из списка)

## 3. Формат пиксельных граней
Каждая грань — массив 8×8 значений Vector3 (RGB).
Пиксели генерируются процедурно и могут быть заменены на текстуры.

## 4. Файловая структура
```
src/
├── Game.cs          — главный цикл, рендер
├── Camera.cs        — камера
├── InputHandler.cs  — ввод
├── Block.cs         — типы блоков, пиксельные грани
├── Chunk.cs         — чанк, генерация меша
├── World.cs         — мир, чанки, рейкаст
└── Program.cs       — точка входа
```
